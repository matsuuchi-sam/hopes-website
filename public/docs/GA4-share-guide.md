# GA4 閲覧権限の共有ガイド

クライアント（サイトオーナー）がGA4のアクセスデータを自分で見れるようにするための手順。
サイト制作後の納品物として、この設定を行う。

## 1. 前提条件

- クライアントが **Googleアカウント**（Gmail）を持っていること
- こちらがGA4プロパティの **管理者権限** を持っていること

> Googleアカウントを持っていない場合は、先に作成してもらう。
> https://accounts.google.com/signup

---

## 2. GA4に閲覧ユーザーを追加する

### 手順

1. https://analytics.google.com/ にログイン
2. 左上のプロパティ選択で、対象サイトのプロパティを選ぶ
3. 左下の **歯車アイコン（管理）** をクリック
4. プロパティ列 →「**プロパティのアクセス管理**」をクリック
5. 右上の「**+**」→「**ユーザーを追加**」
6. 入力：
   - **メールアドレス**: クライアントのGmailアドレス
   - **権限**: 「**閲覧者**」にチェック（他のチェックは外す）
7. 「**追加**」をクリック

### 権限の種類（参考）

| 権限 | できること | 用途 |
|------|-----------|------|
| **閲覧者** | レポートの閲覧のみ | クライアント向け |
| アナリスト | レポート閲覧 + 探索レポート作成 | 分析担当者向け |
| 編集者 | 設定変更が可能 | 運用担当者向け |
| 管理者 | ユーザー管理を含む全権限 | サイト制作者（自分） |

> クライアントには「**閲覧者**」で十分。設定を誤って変更される心配がない。

以下をメールやチャットでそのまま送れるテンプレート。

---

**件名: アクセス解析（GA4）の閲覧方法のご案内**

```
○○様

Webサイトのアクセス解析ツール（Google アナリティクス）の
閲覧権限を設定いたしました。

ご登録いただいたGoogleアカウント（xxxxx@gmail.com）で
以下の方法でアクセスデータをご確認いただけます。

■ スマホの場合
1. 「Google Analytics」アプリをインストール
   - iPhone: https://apps.apple.com/app/google-analytics/id881599038
   - Android: https://play.google.com/store/apps/details?id=com.google.android.apps.giant
2. アプリを開き、登録したGoogleアカウントでログイン
3. プロパティ「○○」を選択

■ PCの場合
1. https://analytics.google.com/ にアクセス
2. 登録したGoogleアカウントでログイン
3. プロパティ「○○」を選択

■ 確認できる主な情報
- サイトへのアクセス数（日別・月別）
- どのページがよく見られているか
- スマホ/PC比率
- お問い合わせフォームの送信数
- アクセス元（検索・SNS・直接アクセス等）

ご不明点がございましたらお気軽にお問い合わせください。
```

## 4. スマホでの閲覧方法

クライアントに案内する際の補足情報。

### アプリのインストール

- **iPhone**: App Store で「Google Analytics」を検索
- **Android**: Google Play で「Google Analytics」を検索

### 初回ログイン

1. アプリを開く
2. 閲覧権限を付与したGoogleアカウントでログイン
3. プロパティ一覧から対象サイトを選択
4. ホーム画面にアクセス数のサマリーが表示される

### 主な画面

| 画面 | 見れること |
|------|-----------|
| ホーム | アクセス数の概要・リアルタイムユーザー数 |
| リアルタイム | 今この瞬間のアクセス状況 |
| ユーザー属性 | 地域・年齢・デバイス |

---

## 5. PCでの閲覧方法

1. https://analytics.google.com/ にアクセス
2. Googleアカウントでログイン
3. 左メニューから各レポートを確認

### よく使うレポート

| メニュー | 内容 |
|---------|------|
| レポート → リアルタイム | 今のアクセス状況 |
| レポート → ユーザー属性 → 概要 | 地域・デバイス・ブラウザ |
| レポート → ライフサイクル → エンゲージメント → ページとスクリーン | ページ別アクセス数 |
| レポート → ライフサイクル → 集客 → トラフィック獲得 | 流入元（検索/SNS/直接） |

## 6. よくある質問

### Q. クライアントが閲覧権限で何かを壊してしまう可能性は？

**ない。** 閲覧者権限ではレポートを見ることしかできない。設定の変更・削除は一切不可。

### Q. 複数人に閲覧権限を付与できる？

**できる。** 手順2を繰り返すだけ。人数制限なし。

### Q. 後から権限を削除したい場合は？

1. GA4管理画面 →「プロパティのアクセス管理」
2. 対象ユーザーの行の右端「×」をクリック
3. 「削除」

### Q. Googleアカウントを持っていないクライアントには？

2つの選択肢：
1. **Googleアカウントを作ってもらう**（推奨）
2. **Looker Studio** でダッシュボードを作り、リンク共有する（Googleアカウント不要）

### Q. Looker Studioとの使い分けは？

| 状況 | 推奨 |
|------|------|
| クライアントが自分で見たい | GA4閲覧権限 |
| 社外の複数人にレポート共有 | Looker Studio |
| 見た目をカスタマイズしたい | Looker Studio |
| 定期レポートをメール送信したい | Looker Studio |
